# Avoid Obstacles Game

A simple Scratch game where the player avoids falling obstacles.

## Game Idea

The player moves left and right using the keyboard arrows.

A ball falls from the top of the screen.

If the ball touches the player, the game ends.

The player wins after surviving and reaching the target score.

## Features

- Left and right player movement
- Random obstacle spawning
- Collision detection
- Score system
- Game Over screen
- Win condition

## Concepts Used

- Events
- Motion
- Variables
- Sensing
- Loops (Forever)
- Conditions (If)

## Controls

- Right Arrow → Move Right
- Left Arrow → Move Left

## Learning Objectives

This project helps children learn:

- Problem Solving
- Computational Thinking
- Event-Driven Programming
- Variables
- Conditional Logic
- Game Development Basics
